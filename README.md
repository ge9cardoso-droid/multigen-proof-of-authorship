## Verification

To independently verify the chronological proof of existence:

```bash
pip install opentimestamps-client
ots verify HASHES_BUNDLE_29-05-2026.txt.ots
```

A successful verification confirms that the documented content existed at or before the Bitcoin block timestamp indicated, without revealing the underlying content.

The OpenTimestamps protocol provides legal evidence of priority of creation, recognized by courts in multiple jurisdictions (France 2025, US FRE 901/902, EU eIDAS 2 Regulation 2024/1183 Article 45l, China 2018 precedent).
